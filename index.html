Markdown语法1

发表于 2018-09-06

标题

Markdown中如果想定义一个标题，需要在标题前加上#。1-6个#分别代表1-6级标题。一般会在#和标题之间加上一个空格。

输出结果：

一级标题

二级标题

三级标题

文本样式

Markdown支持四种文本样式，分别是粗体、斜体、删除线、和粗体加斜体。

粗体

输出结果：
这是粗体文字1
这是粗体文字2

斜体

输出结果：
这是斜体文字1
这是斜体文字2

删除线

输出结果：
这是错误的文本

粗体加斜体

输出结果：
这个文本非常重要
This text is extremely important

引用文字

输出结果：
In the words of Abraham Lincoln:

Pardon my French

引用代码

引用代码有两种形式，

使用一个倒引号`引用代码或命令；

输出结果：
输入命令git status

使用三个倒引号```引用一段代码；

```java
int a = 1;
int b = 2;
int c = a+b;
```

输出结果：

链接

用中括号将链接文本包起来[]，讲URL包在小括号中创建内联链接()。

行内链接示例

[百度](https://baidu.com)

输出结果：

百度

参考链接示例

输出结果：
链接到百度
链接到google
链接到百度2

参考链接中的标号1,2可以是数字，字母，数字和字母的组合。
链接支持相对路径，./表示当前目录，../表示上级目录，不同文档之间的关联采用相对路径。

图片

Markdown使用的图片语法类似于链接的语法，允许两种引入

内联图片

![图片描述文字](../images/icon-next.png)
![图片描述文字](../images/icon-next.png 'next主题')

说明：

  • !
  • [] 图片的属性文本
  • () 包含图片的url或路径，以及title用双引号或单引号括起来的可选属性。

参考引用图片

‘![Alt text][id]’

id是已经定义图片引用的名称。使用与链接引用相同：
[id]: url/to/image "Optional title attribute"

如果需要指定图片的尺寸，可以使用Html 标记：
<img src="../images/icon-next.png" width="50px"/>

自动链接

Markdown支持为URL和电子邮件地址创建“自动”链接的快捷方式：只需用尖括号括起URL或电子邮件地址即可。
<https://baidu.com> <736331518@qq.com>

https://baidu.com 736331518@qq.com

列表

我们可以使用*,+,-,数字作为列表的关键字。同时列表支持嵌套。

  • 服务器端语言
    • java
    • .net
    • delphy
  • 前端语言
    • css
    • html
    • javascript
  • 数据库
    1. sql
    2. mysql
    3. sqlite
    4. oracle
  • 移动端开发(任务列表，非标准Markdown语法)
    • android
    • IOS
    • winphone

使用表情符号

@octocat :+1: This PR looks great - it’s ready to merge!

Atom无效 更多表情符号

段落和换行符

在文本行之间留一个空行来创建新的段落。

忽略Markdown的格式

你可以通过\在Markdown关键字之前使用来转义Markdown关键字。

输出结果：
**转义Markdown关键字**
